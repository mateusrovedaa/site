# Site pessoal

Esse repositório contém o código fonte do website [roveda.dev](roveda.dev), que foi construido utilizando as seguintes ferramentas:
* Hugo
* Netlify
* Github

## Instalação

```
git clone https://github.com/mateusrovedaa/site.git
```

## Uso
```
hugo server -D
```
